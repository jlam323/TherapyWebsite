# PNW Christian Counseling Website

A modern, professional, and calming web application designed specifically for **PNW Christian Counseling**. This site serves as a client connection hub for Marilyn Baker's counseling practice.

## 🌿 Design Philosophy
The website uses a "Sage & Cream" aesthetic to evoke a sense of peace, growth, and professionalism. It is built with a focus on accessibility, mobile responsiveness, and smooth user interactions.

## ✨ Key Features
- **Telehealth Focus**: Clear messaging and CTAs for remote counseling services.
- **Smart Navigation**: A dynamic navigation bar that tracks your progress through the site with a smooth underline indicator.
- **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices.
- **Integrated Contact Hub**: A custom-built contact form and direct links to the Client Portal and Psychology Today.
- **Insurance & Services**: Dedicated sections for insurance providers and specialized counseling focus areas.
- **Real-time Content**: Powered by a central data file for easy updates without touching complex code.

## 🛠 Tech Stack
- **React 18**: For a fast, modern user interface.
- **Tailwind CSS**: For high-performance, utility-first styling.
- **Framer Motion**: For smooth, professional scroll animations and transitions.
- **Lucide Icons**: For clean, consistent visual indicators.
- **Vite**: For lightning-fast development and build times.

## 📝 How to Update Content
The website's content (text, images, navigation links) is managed through a single file: `src/data.json`. 

To update the site:
1. Open `src/data.json`.
2. Modify the text or image paths as needed.
3. The site will automatically reflect these changes.
